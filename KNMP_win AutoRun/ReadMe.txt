KNMP_win
=================================================
                                                 
Instructions:                                    
-------------------------------------------------
1) Copy both "KNMP_win.exe" and "autorun.inf"    
   onto either a blank CD or flash drive.        
2) Insert the loaded CD or flash drive into the
   system on which you wish to execute KNMP_win.
3) Wait a few seconds and you are done!

Notes:
-------------------------------------------------
This method of execution will not work on Win7 or
newer. 

Minimum System Requirements:
-------------------------------------------------
	Hardware:
	-----------------------------------------		
		- Working computer?
		- USB port or optical drive
		- Working audio output
	
	Software:
	-----------------------------------------
		- OS: Windows Vista or Windows XP
		- .NET Framework 2.0 or newer
